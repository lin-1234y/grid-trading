# 网格交易回测工具

## 最简单用法

双击：

```text
启动网格工具.bat
```

浏览器打开后：

1. 输入股票或 ETF 代码，例如 `510300`、`300750`。
2. 点“按代码加载行情”。
3. 填初始现金、网格下限、网格上限、每格买入金额、压力测试目标价。
4. 点“第三步：运行回测”。

## 为什么不是只输入代码就能回测

代码只是名字。回测必须有历史行情。

用 `启动网格工具.bat` 打开时，工具会从：

```text
C:\Users\Dad\Documents\选股系统\data\stock.db
```

读取本地行情。如果某个代码没有数据，需要先在量化分析工具里更新这只股票的行情。

## iPhone 使用

电脑先双击 `启动网格工具.bat`，不要关掉黑色窗口。

iPhone 和电脑连同一个 Wi-Fi，然后用 iPhone 浏览器打开电脑的局域网地址，例如：

```text
http://192.168.1.23:8788
```

## 提醒

这个工具只做研究、回测和资金压力测试，不下单，不连接券商，不构成投资建议。
